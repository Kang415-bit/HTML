เหตุผลที่อยากเรียน Full stack web development

อยากเรียน Full Stack Web Development เพราะอยากมีทักษะที่ครอบคลุมทั้งการพัฒนา Front-end และ Back-end เพื่อให้สามารถสร้างเว็บแอปพลิเคชันได้ครบวงจรด้วยตัวเอง อีกทั้งยังช่วยเพิ่มโอกาสในการทำงานด้านไอทีและต่อยอดไปสู่โครงการหรือธุรกิจส่วนตัวในอนาคตได้

ความคาดหวังต่อวิชานี้

คาดหวังว่าการเรียนวิชา Full Stack Web Development จะทำให้เข้าใจขั้นตอนการพัฒนาเว็บไซต์ทั้งในส่วนของ Front-end และ Back-end อย่างเป็นระบบ ได้ฝึกปฏิบัติจริงผ่านโปรเจกต์ที่ช่วยให้เห็นภาพการทำงานครบวงจร ตั้งแต่การออกแบบหน้าตาเว็บไซต์ การเชื่อมต่อฐานข้อมูล ไปจนถึงการ deploy ให้ใช้งานได้จริง

จุดอ่อน ที่ตนเองต้องพัฒนาเกี่ยวกับทักษะ Programming และ แนวทางการพัฒนา

จุดอ่อนคือการวางโครงสร้างโค้ดที่ยังไม่เป็นระบบ และยังไม่มั่นใจในการเลือกใช้ภาษาและเครื่องมือที่เหมาะสม รวมถึงการแก้ Bug ที่ใช้เวลานาน แนวทางการพัฒนาคือการฝึกเขียนโค้ดตามหลัก Clean Code ศึกษา Best Practices ทดลองทำโปรเจกต์ขนาดเล็กเพื่อเพิ่มประสบการณ์ และฝึกใช้เครื่องมือ Debugger ควบคู่กับการเรียนรู้จาก Community นักพัฒนาเพื่อเสริมทักษะอย่างต่อเนื่อง
